# webcalc
A very simple AngularJS based web calcultor. This calculator only works on integer numbers.

## Application structure

```
.
├── README.md
├── app
│   ├── js
│   │   └── webcalc.js
│   ├── style
│   │   └── webcalc.css
│   └── test
└── webcalc.html
```
